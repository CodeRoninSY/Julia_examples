# How to add package via github url:<br>
You can either: <br>
<p>
 <code>] add https://github.com/TuringLang/Libtask.jl</code><br>
</p>
or equivalently:<br>
<p>
 <code>using Pkg</code><br>
 <code>Pkg.add(PackageSpec(url="https://github.com/TuringLang/Libtask.jl")</code><br>
